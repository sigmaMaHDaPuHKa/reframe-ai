// architecture

Пайплайн обработки

step_1

Кодирование HEVC (H.265)

Видеопоток сжимается кодеком с агрессивным CRF. Размер файла падает в 5-10x. Побочный эффект — блочные артефакты, потеря высокочастотных деталей, цветовой banding.

step_2

ESRGAN inference

Enhanced Super-Resolution GAN анализирует каждый кадр. Генератор восстанавливает текстуры и края. Дискриминатор обеспечивает реалистичность. Работает покадрово через TensorFlow.js на GPU клиента.

step_3

Результат

Файл весит как сжатый (2.1 MB вместо 12.4 MB), а визуальное качество на уровне оригинала. VMAF > 90. Laplacian variance увеличивается на 15-40% относительно сжатого кадра.

input.mp4 → HEVC encode → compressed.mp4 → ESRGAN → output.mp4
