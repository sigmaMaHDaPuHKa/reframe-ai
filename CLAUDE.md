# 🎮 REQUIRED INTERACTIVE CONCEPTS

The project MUST include at least 2–3 of these:

### 1. "Find the Difference" Game
- Show two videos (A and B)
- One is normal compression
- One is AI-enhanced
- User must guess which is better
- Reveal result → surprise effect

### 2. "Restore the Destroyed Video"
- Show extremely compressed ugly video
- Button: "Fix with AI"
- Transition → clean version
- Should feel like magic

### 3. Bitrate vs Quality Slider
- Slider changes:
  - video quality
  - file size
- Toggle: with AI / without AI
- Show how AI keeps quality at lower bitrate
